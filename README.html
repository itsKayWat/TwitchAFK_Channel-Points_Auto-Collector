TwitchAFK - Channel Points Auto-Collector

Purpose

TwitchAFK allows you to automatically collect channel points from multiple Twitch streams simultaneously without having to actively watch them. It runs in the background and integrates with a Chrome extension for easy point management.

Why Use TwitchAFK?

  • Collect channel points while you're away
  • Monitor multiple streams simultaneously
  • Track point accumulation through a clean interface
  • Save time and maximize point collection efficiency
  • Never miss a point claim opportunity

Requirements

  • Python 3.7+
  • Google Chrome browser
  • Windows/Mac/Linux operating system

Installation


1. Clone the repository
git clone https://github.com/yourusername/TwitchAFK.git

2. Install requirements
python requirements.py

3. Load the Chrome extension:
   - Open Chrome
   - Go to chrome://extensions/
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the "twitch_points_manager" folder

Usage Example

Imagine you want to collect channel points from your favorite streamers to unlock channel-specific rewards, but you can't watch streams all day. With TwitchAFK:

  1. Start the Python script
  2. Log in to your Twitch account when prompted
  3. Add your favorite channels through the extension
  4. Let it run in the background while you work or sleep
  5. Check your accumulated points through the extension
